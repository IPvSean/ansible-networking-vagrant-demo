# Ansible Networking Training Course

1. Connect to the ansible control node

   ```bash
   vagrant ssh ansible
   ```

2. Change directories into the exercise you want to complete

   ```bash
   cd ansible-networking-vagrant-demo/training-course/exercise01
   ```
